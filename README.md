# Smart Search
Moteur de recherche - scanner - whois - google maps - information

# Téléchargement

git clone https://github.com/VlainLom/SmartSearch.git

# Installation

cd SmartSearch

pip3 install selenium

pip3 install art

pip3 install termcolor

pip3 install requests

pip3 install webbrowser

pip3 install bs4

# Exécution

python3 main.py
